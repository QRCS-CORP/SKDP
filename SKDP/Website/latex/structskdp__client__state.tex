\doxysubsection{Detailed Description}
The SKDP client state structure. 

This structure holds the state information for an SKDP client during a key exchange and ongoing communication session. It contains\+:


\begin{DoxyItemize}
\item {\ttfamily rxcpr\+:} The receive channel cipher state.
\item {\ttfamily txcpr\+:} The transmit channel cipher state.
\item {\ttfamily ddk\+:} The device derivation key.
\item {\ttfamily dsh\+:} The device session hash, computed from the device identity, configuration, and a random token.
\item {\ttfamily kid\+:} The device identity string.
\item {\ttfamily ssh\+:} The server session hash received during the key exchange.
\item {\ttfamily expiration\+:} The expiration time for the current session (in seconds from epoch).
\item {\ttfamily rxseq\+:} The receive channel packet sequence number.
\item {\ttfamily txseq\+:} The transmit channel packet sequence number.
\item {\ttfamily exflag\+:} A flag indicating the progress/status of the key exchange. 
\end{DoxyItemize}
